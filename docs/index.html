scScale

Gaussian-spike scaling laws for matched single-cell modalities, using one low-rank subspace-alignment mutual information formula.

Minimal RNA-ADT Tutorial UMI scaling, cell-number scaling, joint scaling, and empirical MI. Package Manual PDF Function reference for the current package API. Source Code Install and inspect the package on GitHub.

Core Workflow

library(scScale)

pair <- scscale_pair_fit(rna_counts, adt_counts)

umi <- scscale_umi_mi(pair, rna_counts, sampling_rates = c(0.25, 0.5, 1))
cells <- scscale_cell_number_mi(pair, n_grid = c(500, 1000, 2000, 4000))
joint <- scscale_cell_number_by_umi_mi(pair, umi, n_grid = c(500, 1000, 2000, 4000))

empirical <- scscale_empirical_mi(rna_counts, adt_counts)
