`scScale` contains small R utilities for single-cell scaling-law analyses:
spectral mutual information from eigenvalues, matrix-based batch-effect
information, and batch-effect scaling-law fits.

Start with one of the focused tutorials:

- [Cell number scaling law](cell-number-scaling-law.html)
- [UMI scaling law](umi-scaling-law.html)
- [Batch number scaling law](batch-number-scaling-law.html)

Each tutorial starts from a compact real Jurkat 10x count block already present
in this repository, then keeps the computation small enough to render quickly.
